# 指数函数

注意：下列函数签名中 T 可表示 float, vec2, vec3, vec4

# pow

  • 签名：T pow(T x, T y)
  • 作用：xy

# exp

  • 签名：T exp(T x)
  • 作用：ex

# log

  • 签名：T log(T x)
  • 作用：ln(x)

# exp2

  • 签名：T exp2(T x)
  • 作用：2x

# log2

  • 签名：T log2(T x)
  • 作用：log2(x)

# sqrt

  • 签名：T sqrt(T x)
  • 作用：平方根

# inversesqrt

  • 签名：T inversesqrt(T x)
  • 作用：平方根倒数
